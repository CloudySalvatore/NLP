# NLP
第一题：
首先将构建所有文本的词典。
然后统计数据集中最长句子的字数。
再将每个句子填充到此数据集中最大句子字数（236字）。
然后将每个字通过词典转化为相应的词向量。
通过双向LSTM模型，对每个字生成每个类别的概率，再经过CRF，确定最终类别结果。
整个训练过程是将一个句字的每一个字输入模型训练，输出每一个字的类别。



第二题：
首先对每个句子行进中文实体识别，将识别出来的实体进行标注。
然后将每个句子的每个字通过词典转化为相应的词向量。
将标注的实体与句子一起放入模型中训练。
然后通过单向LSTM模型，产生实体之间的每个类别的概率，再经过一层全连接神经网络，确定最终类别结果。
